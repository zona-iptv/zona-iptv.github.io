HLS Player - m3u8 Streaming Player

HLS Player a chrome extention which implements an HTTP Live Streaming client. You can play .m3u8 HTTP Live Streaming (HLS) . It relies on HTML5 video and MediaSource Extensions for playback. It works by transmuxing MPEG-2 Transport Stream and AAC/MP3 streams into ISO BMFF (MP4) fragments. This transmuxing could be performed asynchronously using Web Worker if available in the browser. It also supports HLS + fmp4, as announced during WWDC2016
